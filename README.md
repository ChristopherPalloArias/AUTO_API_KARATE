# QA Automation Challenge - Karate Framework

## Propósito
Este repositorio contiene la base inicial limpia para el reto técnico de QA del Taller Semana 7, implementado con enfoque AI First.

## Stack Tecnológico
- Java
- Maven
- Karate Framework

## Ejecución
Para ejecutar las pruebas del proyecto, utiliza el siguiente comando:

```bash
mvn test
```

> **Nota:** Los escenarios y la lógica de pruebas (`.feature`) se agregarán en una fase posterior. El proyecto está preparado para su ejecución una vez implementados.
